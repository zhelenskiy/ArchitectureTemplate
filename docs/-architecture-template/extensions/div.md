# div  
[jvm]  
Brief description  
Redefines div operator for [BigInteger](https://docs.oracle.com/javase/8/docs/api/java/math/BigInteger.html) and [BigInteger](https://docs.oracle.com/javase/8/docs/api/java/math/BigInteger.html) instances. The purpose of the redefinition is to resolve [BigInteger] / [BigInteger] when [BigInteger] / [Number] and [Number] / [BigInteger] are defined.  
  


#### Return  
Resulting [BigInteger](https://docs.oracle.com/javase/8/docs/api/java/math/BigInteger.html)
