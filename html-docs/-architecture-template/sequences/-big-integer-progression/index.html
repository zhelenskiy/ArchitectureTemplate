equals
Link copied to clipboard
Two BigIntegerProgressions are equal when they are equal as sequences.This means:

  • If other is not BigIntegerProgression, equality is checked as for Sequence

  • Empty BigIntegerProgressions are equal only to each other

  • Single ranges are equal when their only elements are equal

  • Other ones are equal when the sequences have equal first, last, step.
