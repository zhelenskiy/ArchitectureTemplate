SingleRange

The class represents BigIntegerProgression, sequences.BigIntegerRange with exact 1 element.All methods here are just to specify return type (EmptyRange) statically.Method toString works as it is defined in BigIntegerProgression.Step equals to 1 for all instances of SingleRange.
data class SingleRange(only: BigInteger) : BigIntegerRange
